# Calculator

A calculator built using HTML, CSS, and JavaScript.

## Features

- Keyboard support
- Live clock and date
- Glitter cursor effect
- Basic arithmetic operations
- Clean and modern design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Future Improvements

- Scientific mode
- Calculation history
- Theme switcher
- Memory buttons
